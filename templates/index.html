How to use the Function Box in Problem Solvers:

Normally, you can use (**) for powers. Eg: 23 would be represented as 2**3 in the function box

You can use exp() to substitute for e in your expressions. For exmaple, ex would be represnted as exp(x)

Enforce parantheses where needed in order to prevent unwanted BODMAS evaluation.
